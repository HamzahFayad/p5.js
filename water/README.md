Interactions:

- Mouse Hovering
- Mouse Double Click (to shrink particles)
- Spacebar (to reset shrinking size)

- UP_ARROW (ice crystals form)
- DOWN_ARROW (fluid form)
- LEFT_ARROW (flying bird)

- Keys 1 to 8 (create sin curve like wavy forms // different "amplitudes")

- F (show Framerate)
- I (show possible Interactions)

- enable/disable audio on top left icon
- switch day/night mode on top right icon
